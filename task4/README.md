### Рассматриваемое уравнение

Будем решать уравнение:  $y'' - \lambda y = - 2 \lambda \sin (\sqrt \lambda x)$ с граничными условиями Дирихле $y(0) = 0$, $y(4 * \pi / \sqrt{λ}) = 0$.
Данное уравнение имеет решение вида: $y = sin(\sqrt{λ}*x)$
